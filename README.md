# Foodica Sales Person Analytics Dashboard

This is a Next.js Web App version of the Foodica dashboard.

## What is included
- Arabic / English toggle.
- Dark Mode / Light Mode toggle.
- Foodica logo inside the app.
- CSV Export enabled.
- Server-side API routes for analytics and CSV export.
- Security headers and no-index headers.
- Footer:
  - @Foodica 2026
  - by Kareem Shehata

## Important security note
This is stronger than a single static HTML file because the analytics data is served through API routes and the full backend source is not delivered by Save As.

Because you requested no username/password, anyone with the deployed link can open the dashboard. Also, anything visible in the browser can still be copied, screenshotted, or inspected by advanced users.

## Local run
```bash
npm install
npm run dev
```

Open:
```text
http://localhost:3000
```

## Deploy to Vercel
1. Create a new GitHub repository.
2. Upload these project files to the repository.
3. Open Vercel.
4. Choose Add New Project.
5. Import the GitHub repository.
6. Keep the default Next.js settings.
7. Click Deploy.
